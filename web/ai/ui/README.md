## Conventions

- One file per component or closely related component family.
- File name matches the component file in `src/components/ui/`.
- Update the component doc when changing component API, variants, anatomy, or gotchas.
- General wrapper rules live in [`../baseui`](../baseui/) and root-level [`../tailwind-variants.md`](../tailwind-variants.md).
